## HW02
### 爬取網站
- TAIN-桌球站 球員數據
- [資料網址](https://tain.tw/stats?type=2&points_page=1)
### 爬取內容
將每一頁的球員數據爬取下來，包含以下欄位：
- Pos
- 球員
- 排名
- 勝率
- 直板勝率
- 橫板勝率
- 右手勝率
- 左手勝率
- 男子勝率
- 女子勝率
- 穩定率
- 爆發率
- 平面勝率
- 短顆勝率
- 中顆勝率
- 長顆勝率
- NT勝率
